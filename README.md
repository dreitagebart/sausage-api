# Sausage API

## usage

This is a next.js typescript based web API. You can deploy it on [vercel.com](https://vercel.com).

## routes

`/api/random`

returns a random sausage

`/api/all`

returns all stored sausages

`/api/search/:term`

search for a specific sausage where ":term" is your search term. E.g. /api/search/fleisch returns all sausage includung the word "fleisch" like "Fleischk채se", "Gebackener Fleischk채se", "Grober Fleischk채se", "Kalbfleischk채se", etc.
